1. 代数基础

主要包括整式与分式、方程与不等式、复数、指数与对数等。

方程与不等式

一次、二次方程的求解；二次方程判别式 $\\Delta=b^2-4ac$：若 $\\Delta>0$ 有两个不等实根；$\\Delta=0$ 有重根；$\\Delta<0$ 无实根。

二次公式：$x=\\dfrac{-b\\pm\\sqrt{b^2-4ac}}{2a}$
示例：解方程 $2x^2-3x-5=0$
判别式 $\\Delta=(-3)^2-4\\cdot2\\cdot(-5)=9+40=49$，则 $x=\\dfrac{3\\pm7}{4}$，得到 $x= \\dfrac{10}{4}=\\dfrac{5}{2}$ 或 $x=-1$。

分式与恒等变形

约分、通分、化简，注意分母不能为 0。解分式方程时需检查去分母后是否产生增根。

复数

复数常写为 $a+bi$，$i^2=-1$。模与幅角：$|z|=\\sqrt{a^2+b^2}$。

2. 函数与图像

函数的定义、奇偶性、单调性、最大最小值、周期性、初等函数（多项式、分式、指数、对数、幂、三角）以及复合、反函数。

常见函数性质

  • 奇函数：$f(-x)=-f(x)$，关于原点对称；偶函数：$f(-x)=f(x)$，关于 $y$ 轴对称。
  • 单调性通过导数判断：若 $f'(x)>0$ 则单调递增（高考常考）。
示例：求 $f(x)=x^3-3x$ 的单调区间
$f'(x)=3x^2-3=3(x^2-1)$，解 $f'(x)=0$ 得 $x=\\pm1$。因此 在 $(-\\infty,-1)$ 和 $(1,\\infty)$ 上 $f'(x)>0$（递增），在 $(-1,1)$ 上 $f'(x)<0$（递减）。

函数图像与平移伸缩

例如 $y=f(x-a)+b$ 表示图像向右平移 $a$，向上平移 $b$。$y=k f(x)$：竖直伸缩。

3. 几何（平面与向量）

三角形、平行线、圆、相似与全等、坐标几何、向量基础与点积、线段比与面积比等。

三角形重要结论

  • 内角和：$\\alpha+\\beta+\\gamma=180^\\circ$。
  • 余弦定理：$c^2=a^2+b^2-2ab\\cos C$（任意三角形）。
  • 面积公式：$S=\\dfrac12 ab\\sin C$。
示例：已知三角形两边 $a=5,b=7$，夹角 $C=60^\\circ$，求第三边 $c$
由余弦定理：$c^2=5^2+7^2-2\\cdot5\\cdot7\\cos60^\\circ=25+49-70\\cdot\\frac12=74-35=39$，所以 $c=\\sqrt{39}$。

向量与坐标几何

向量的线性运算、点积（内积）和投影：$\\vec{a}\\cdot\\vec{b}=|\\vec a||\\vec b|\\cos\\theta$；点积为 0 表示垂直。

4. 三角函数与公式

正弦、余弦、正切函数及其周期、诱导公式、和差化积、倍角与半角公式。

主要公式示例：
$ \\sin(A\\pm B)=\\sin A\\cos B\\pm\\cos A\\sin B$
$ \\cos(A\\pm B)=\\cos A\\cos B\\mp\\sin A\\sin B$
$ \\sin 2A=2\\sin A\\cos A,\\quad \\cos 2A=\\cos^2A-\\sin^2A$
示例：化简 $\\sin x + \\sin 3x$
使用和化积：$\\sin x+\\sin 3x=2\\sin(2x)\\cos(x)$.（因为 $\\sin u+\\sin v=2\\sin\\frac{u+v}{2}\\cos\\frac{u-v}{2}$）

5. 数列（等差、等比与递推）

等差数列：$a_n=a_1+(n-1)d$；前 n 项和 $S_n=\\dfrac{n}{2}(a_1+a_n)$. 等比数列：$a_n=a_1 r^{n-1}$；若 $r\\neq1$，$S_n=a_1\\dfrac{1-r^n}{1-r}$。

示例：等比数列首项 3，公比 2，求前 4 项和
$S_4=3\\cdot\\dfrac{1-2^4}{1-2}=3\\cdot\\dfrac{1-16}{-1}=3\\cdot15=45$。

递推数列（常见方法）

线性递推可用特征方程法，需掌握常见的化简与归纳证明。

6. 解析（极限、导数、积分简介）

这是微积分的入门，重视极限与连续性、导数的定义与应用、简单积分的几何意义。

极限

极限的基本操作与常见极限：$\\lim_{x\\to0}\\dfrac{\\sin x}{x}=1$。

导数

导数定义：$f'(x)=\\lim_{h\\to0}\\dfrac{f(x+h)-f(x)}{h}$. 常见导数：$(x^n)'=nx^{n-1}$，$(\\sin x)'=\\cos x$ 等。

积分（定积分）

定积分表示面积：$\\int_a^b f(x)\\,dx$，基本原理为牛顿-莱布尼茨公式：$\\int_a^b f(x)\\,dx=F(b)-F(a)$，其中 $F'=f$。

示例：计算 $\\dfrac{d}{dx}(x^2\\sin x)$
使用乘积法则：$(x^2)' \\sin x + x^2(\\sin x)' = 2x\\sin x + x^2\\cos x$。

7. 概率与统计

概率的加法、乘法规则、条件概率与全概率公式、贝叶斯公式；统计上的均值、中位数、方差、标准差、常见分布（离散的二项分布、连续的正态分布概念）。

条件概率：$P(A|B)=\\dfrac{P(A\\cap B)}{P(B)}$；全概率：$P(A)=\\sum_i P(A|B_i)P(B_i)$。
示例：两个独立事件 A、B，已知 $P(A)=0.3, P(B)=0.4$，求 $P(A\\cap B)$
若独立，则 $P(A\\cap B)=P(A)P(B)=0.3\\times0.4=0.12$。

8. 线性代数基础（矩阵与行列式）

矩阵的加法、乘法、逆矩阵（方阵且行列式非零时存在逆矩阵）、行列式与其性质、线性方程组的矩阵表示（高斯消元）。

示例：2x2 矩阵逆矩阵
若 $A=\\begin{pmatrix}a&b\\\\c&d\\end{pmatrix}$，且 $\\det A=ad-bc\\neq0$，则 $A^{-1}=\\dfrac{1}{ad-bc}\\begin{pmatrix}d&-b\\\\-c&a\\end{pmatrix}$。

解题思路与常见技巧

  • 审题：标清已知与���知、条件的量纲与约束（如正负、整数、非零）。
  • 画图：几何题画图通常能揭示关系，函数题画图查看单调/零点/极值。
  • 代换与函数变形：合适代换（如 $t=\\tan\\frac{x}{2}$）能化简三角表达式。
  • 边界与极限思路：讨论问题通常需要考虑极限、临界点与边界情况。
  • 归纳与构造反例：证明问题常用归纳或反证法；判定命题非真时找反例。
